# Harika Vaddadi

A personal website showcasing:

* AI Engineering Projects
* Poetry
* Recipes
* Journal Entries
* Resume

Built with:

* Astro
* Tailwind CSS
* Markdown Content
* GitHub Pages

## Features

### AI Projects

Document real-world AI solutions:

* RAG Systems
* Insurance Entity Extraction
* Demand Packet Assistant
* Computer Vision Projects
* MLOps Work

### Poetry

Poetry exploring:

* Technology
* Humanity
* Coffee
* Sunsets
* Beaches

### Recipes

Healthy and simple recipes.

### Journal

Thoughts on:

* AI
* Life
* Productivity
* Walking

* Learning

## Local Development

Install dependencies:

```bash
npm install
```

Start development server:

```bash
npm run dev
```

Open:

http://localhost:4321

---

## Build

```bash
npm run build
```

---

## Preview Production Build

```bash
npm run preview
```

---

## Deploy

Push code to GitHub.

GitHub Actions automatically deploys to GitHub Pages.

Website URL:

```
https://<github-username>.github.io/
```

or

```
https://your-domain.com
```

---

## Content Creation

Add markdown files under:

```
src/content/projects
src/content/poems
src/content/recipes
src/content/journal
```

Example:

```
src/content/poems/coffee-and-code.md
```
